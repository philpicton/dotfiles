# AeroSpace Keyboard Shortcuts

## Terminal & Applications

- `Alt + Enter` - Open Kitty terminal

## Window Focus (Vim-style)

- `Alt + H` - Focus left
- `Alt + J` - Focus down
- `Alt + K` - Focus up
- `Alt + L` - Focus right

## Move Window

- `Alt + Shift + H/J/K/L` - Move window (vim keys)
- `Alt + Shift + ←/→/↑/↓` - Move window (arrows)

## Window Layout

- `Alt + /` - Toggle tiles layout (horizontal/vertical)
- `Alt + ,` - Toggle accordion layout (horizontal/vertical)
- `Alt + F` - Toggle floating/tiling mode
- `Alt + Shift + Enter` - Fullscreen

## Window Resize

- `Alt + -` - Decrease size (smart resize -50)
- `Alt + =` - Increase size (smart resize +50)

## Workspaces

- `Alt + 1-9` - Switch to workspace 1-9
- `Alt + Shift + 1-9` - Move window to workspace 1-9 (follows focus)
- `Alt + Tab` - Switch to previous workspace
- `Alt + ←/→` - Previous/next workspace (wrap around)
- `Ctrl + ←/→` - Previous/next workspace (wrap around)

## Multi-Monitor

- `Ctrl + Alt + ←/→` - Move workspace to previous/next monitor

## Service Mode

- `Alt + Shift + ;` - Enter service mode

### Service Mode Commands

- `Esc` - Reload config and exit to main mode
- `R` - Reset layout (flatten workspace tree)
- `F` - Toggle floating/tiling
- `Backspace` - Close all windows except current
- `Alt + Q` - Disable AeroSpace
- `Alt + Shift + H/J/K/L` - Join window with adjacent window
